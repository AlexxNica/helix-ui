A panel is a utility for sectioning content vertically into a head, body, and foot. A panel will consist of a head, body, and foot.

  • The <hx-panelhead> element is fixed to the top of the panel.
    • This element is optional.
  • The <hx-panelbody> element fills the space between the head and foot.
  • The <hx-panelfoot> element is fixed to the bottom of the panel.
    • This element is optional.

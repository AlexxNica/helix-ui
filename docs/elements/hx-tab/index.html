The custom <hx-tab> element labels a companion <hx-tabpanel> element.

Permitted Parents an <hx-tablist> element Permitted Children Any
{% endblock %} {% block attributes %}
aria-controls
Automatically set by <hx-tabset> when attached to the DOM.
aria-selected
Automatically set as the current attribute/property changes.
current
Automatically applied by <hx-tabset> to identify the current tab in a tablist.
id
Used by <hx-tabset> to wire up a tab to its tabpanel using the aria-controls and aria-labelledby attributes. If you do not define an id attribute, one will be automatically generated.
role
Automatically set, if not explicitly defined in markup.
